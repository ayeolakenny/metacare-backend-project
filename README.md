## Installation 

1. **Clone** the project. 

   ```bash
   git clone https://github.com/ayeolakenny/metacare-backend-project
   ```

2. **Install** packages. 

   ```bash
   yarn
   ```
3. **Migrate dev**

   ```bash
    yarn migrate --name metacare
   ```
   
4. **Seed DB**

   ```bash
     yarn db:seed
   ```
   
5. **Generate Prisma**

   ```bash
     yarn generate
   ```

6. **Run** the server

   ```bash
   yarn start:dev
   ```
